Tools For Recon using  https://hackertarget.com API

## Installation
```bash
Chmod 777 r3con.sh
./r3con.sh
```

## Usage

```bash
=========================================
Usage : 
-h help
-s Subdomain Scan
-d DNS lookup
-w WhoIs
-r Reverse DNS
-a Check HTTP Headers
-c Completer Scan
========================================= 

```
